# Sum All Digits
You are given an integer n. Return the sum of its digits.

**Example**

For n = 111, the output should be
addTwoDigits(n) = 3.

**Input/Output**

- **[input] integer n**

    A positive two-digit integer.

*Guaranteed constraints:*

1 ≤ n ≤ 9999.

- **[output] integer**

    The sum of all digits of the input number.
